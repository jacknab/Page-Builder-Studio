Your VPS is the single entry point

You host ALL client sites on one VPS (or load-balanced cluster later).

Example:

VPS IP: 123.45.67.89
Your system (Certxa) routes traffic internally based on domain name

So the DNS only ever points to your VPS.

🌐 2. What the client does (domain side)

They go into their domain registrar (GoDaddy, Namecheap, etc.)

They add an A record like this:

Type: A
Host: @
Value: 123.45.67.89
TTL: 300 (or default)

Optional but recommended:

Type: A
Host: www
Value: 123.45.67.89

That’s it.

Now:

clientdomain.com → your VPS
www.clientdomain.com → your VPS
🧭 3. What YOUR VPS must do (important part)

This is where your platform becomes “smart hosting”.

You need to detect the domain and serve the correct site.

If you're using Nginx (recommended):

You use a wildcard or catch-all server block:

server {
    listen 80;
    server_name _;

    root /var/www/certxa-sites;

    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}

But the key idea is:

👉 You read the Host header:

Host: clientdomain.com

Then your backend decides:

load client A’s site
or client B’s site
🧠 4. Your backend routing logic (this is Certxa magic)

Inside your Node/Next/Vite backend:

Pseudo logic:

const host = req.headers.host;

const site = await db.sites.findOne({
  domain: host
});

if (!site) return res.status(404).send("Not found");

renderSite(site);

So every domain maps to a database entry:

domain
theme
content
booking config
staff etc.
🔐 5. SSL (don’t skip this)

You’ll want HTTPS per domain.

Use Let’s Encrypt + Certbot

If you want scale later:

use wildcard SSL OR
auto-generate certs per domain

Simple setup:

certbot --nginx -d clientdomain.com -d www.clientdomain.com

Or automate it in your platform (better long-term).

🚀 6. What your system becomes

You’re basically building:

VPS = hosting layer
Certxa backend = router + CMS engine
Domain = just a pointer (A record)

So every client is just:

Domain → VPS → Certxa → correct site
